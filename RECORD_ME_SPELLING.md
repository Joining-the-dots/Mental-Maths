# RECORD ME — Spellcraft words (359 words)

One recording session (or several — any split is fine). For **each word**, read the number as a marker, then say the word **twice, clearly**, with a short pause between:

> "Number one. …the. …the. Number two. …do. …do." 

Say the number in words (it's only a splitting marker — it gets cut out). Speak the word the way you'd say it to the kids, natural and clear. If you fluff one, just repeat the same number again and carry on — the last take of each number wins. Drop the finished recording(s) into OneDrive → Times tables → recordings, and everything gets split, named and wired in from there.
